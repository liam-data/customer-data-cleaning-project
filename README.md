# Customer Data Cleaning Project

## Overview
This project demonstrates a data cleaning workflow using Python and pandas. It takes a messy customer dataset and transforms it into a clean, standardized format suitable for analysis or operational use.

## Skills Demonstrated
- Python
- pandas
- Data cleaning
- Data validation
- Removing duplicates
- Handling missing values
- Data formatting and standardization

## Files
- `customer_data.csv` - Raw dataset
- `clean_data.py` - Script used to clean the data
- `cleaned_customer_data.csv` - Final cleaned dataset

## What the Script Does
- Removes duplicate records
- Cleans and formats names
- Standardizes email addresses
- Identifies missing or invalid emails
- Formats phone numbers consistently
- Standardizes city names
- Fills missing values
- Outputs a cleaned dataset

## Why This Project Matters
Data quality is critical in both technical and business environments. This project shows the ability to take messy, real-world data and turn it into something usable and reliable.

## How to Run
1. Install Python
2. Install pandas with `pip install pandas`
3. Run the script using `python clean_data.py`
